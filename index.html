O que é um jogo?

Jogos

  • São jogados voluntariamente
  • Tem objetivos
  • Tem regras
  • Podem ter vitória e derrota
  • São interativos
  • Tem desafios
  • Podem criar valores internos próprios
  • Envolvem os jogadores (imersão)
  • São sistemas fechados, formais
  • Fechado: o sistema tem limites
  • Formal: regras bem definidas

Um jogo é uma atividade de solução de problemas, encarado de forma lúdica.

- Jesse Schell, A arte de Game Design
